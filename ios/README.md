# ARUS iOS Platform

This directory contains the native iOS/iPadOS application generated by Capacitor.

## Quick Reference

### Commands (run from project root)

```bash
# Build frontend and sync to iOS
npm run build:renderer && npx cap sync ios

# Open in Xcode (macOS only)
npx cap open ios

# Just sync assets (after code changes)
npx cap sync ios
```

### Building for iPad

See [../docs/IOS_BUILD_GUIDE.md](../docs/IOS_BUILD_GUIDE.md) for complete instructions.

**Prerequisites (macOS only):**

1. Install Xcode 16+
2. Install CocoaPods: `sudo gem install cocoapods`
3. Run: `cd ios/App && pod install`

**Build Steps:**

1. Build frontend: `npm run build:renderer`
2. Sync to iOS: `npx cap sync ios`
3. Open Xcode: `npx cap open ios`
4. Select your team under Signing & Capabilities
5. Choose iPad device/simulator
6. Click Run (▶️)

### Troubleshooting

**CocoaPods not installed:**

```bash
sudo gem install cocoapods
cd ios/App
pod install
```

**Build errors in Xcode:**

```bash
# Clean and rebuild
cd ios/App
pod install
# Then in Xcode: Product → Clean Build Folder
```

**Web assets not updating:**

```bash
npm run build:renderer
npx cap sync ios
# Re-run in Xcode
```
