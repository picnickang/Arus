==================================================
MANDATORY ARCHITECTURE REQUIREMENTS
==================================================

Hexagonal modularity must be preserved.

Do not place business logic directly inside React components.

Follow this structure where applicable:

- domain:
  role/menu/navigation models and types

- application:
  role-to-navigation mapping
  visibility rules
  navigation orchestration

- infrastructure:
  adapters to auth/session/feature flags

- interfaces:
  React components
  layouts
  route views

Frontend architecture rules:
- Components render state; they do not decide policy.
- Role visibility logic must live in typed application/navigation services.
- Reusable navigation config must be centralized.
- Avoid duplicated role checks scattered across components.
- Preserve backend RBAC enforcement; UI visibility is usability only, not security.
