C. HMAC Validation Scope

Keep HMAC middleware only on telemetry ingest/heartbeat style routes. Confirm body is signed against the raw body:

F. Observability & Errors

Wrap every async handler with a shared asyncHandler(fn) to avoid unhandled promise rejections and rely on the central error middleware:

const asyncHandler = <T extends express.RequestHandler>(fn: T): T =>
  ( (req,res,next) => Promise.resolve(fn(req,res,next)).catch(next) ) as T;


Replace inline try/catch blocks with asyncHandler where feasible. Keep explicit try/catch only when you want to return custom 4xx with Zod details.
