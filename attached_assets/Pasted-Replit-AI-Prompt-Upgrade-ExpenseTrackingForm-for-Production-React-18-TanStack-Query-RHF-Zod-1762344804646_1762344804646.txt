Safe Numeric Handling

Introduce a numberChange helper:

const numberChange = (onChange:(v:number)=>void) =>
  (e:React.ChangeEvent<HTMLInputElement>) => {
    const v = e.currentTarget.valueAsNumber;
    onChange(Number.isFinite(v) ? v : NaN);
  };


Use it for all number inputs (amount, hourlyLossRate, etc.) instead of parseFloat(...) || 0.
