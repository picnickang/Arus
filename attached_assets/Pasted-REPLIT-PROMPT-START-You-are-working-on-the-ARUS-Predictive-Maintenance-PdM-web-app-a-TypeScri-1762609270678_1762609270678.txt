3. Backend API (

server/routes/sensorTemplates.ts

)

Add support for filtered queries:

router.get("/", async (req, res) => {
  const { equipmentType } = req.query;
  const query = db.select().from(sensorTemplates);
  if (equipmentType) {
    query.where(eq(sensorTemplates.equipmentType, equipmentType));
  }
  const templates = await query;
  res.json(templates);
});
This enables the frontend to fetch only templates relevant to that equipment type.
