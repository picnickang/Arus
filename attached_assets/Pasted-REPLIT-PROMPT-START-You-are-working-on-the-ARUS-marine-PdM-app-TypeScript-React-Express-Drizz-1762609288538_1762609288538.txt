// POST /api/equipment/:equipmentId/apply-sensor-bundle
// body: { bundleId: string }

- For each `templateId` in the bundle:
  - Load the corresponding `sensorTemplates` row.
  - Create a new `sensor_config` row using template defaults:

    - equipmentId
    - sensorType (from template.kind)
    - unit (from template.unit)
