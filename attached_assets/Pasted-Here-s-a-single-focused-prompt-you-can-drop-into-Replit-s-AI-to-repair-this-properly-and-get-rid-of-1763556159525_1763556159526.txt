  `fs18.existsSync is not a function`

- This is incorrect, because `fs.promises` does NOT have `existsSync`.

- In the ORIGINAL TypeScript source:

    import * as fs from 'fs';

  - Use that `fs.existsSync()` for the static root checks.

- If you also need async fs operations elsewhere, do this cleanly:

  ```ts

Make sure there is NO spot in the code where fs.promises is accidentally used with .existsSync(). Any existsSync call must be on the sync fs module.
