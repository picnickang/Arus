In your RulEngine.calculateRul(...) (after you compute remainingDays, failureProbability, confidenceScore, healthIndex, riskLevel, opMode, and q), record metrics:

import { rulCalcDuration, rulRemainingDays, rulFailureProb, rulConfidence, rulQuality, rulHealthIdx, rulRiskLevel } from "./observability/rul-metrics.js";

const start = Date.now();
// ... your RUL logic ...
const duration = Date.now() - start;

const labels = { org_id: orgId, equipment_id: equipmentId, mode: opMode };

rulCalcDuration.labels(labels.org_id, labels.equipment_id, labels.mode).observe(duration);
rulRemainingDays.labels(labels.org_id, labels.equipment_id, labels.mode).set(remainingDays);
rulFailureProb.labels(labels.org_id, labels.equipment_id, labels.mode).set(failureProbability);
rulConfidence.labels(labels.org_id, labels.equipment_id, labels.mode).set(confidenceScore);
rulQuality.labels(labels.org_id, labels.equipment_id, labels.mode).set(q);
rulHealthIdx.labels(labels.org_id, labels.equipment_id, labels.mode).set(healthIndex);

const riskNum = ({low:0, medium:1, high:2, critical:3} as any)[riskLevel] ?? 0;
rulRiskLevel.labels(labels.org_id, labels.equipment_id, labels.mode).set(riskNum);
Expose metrics (if not already):

// server/observability/metrics-route.ts
import client from "prom-client";
import { Router } from "express";

export const metricsRouter = Router();
client.collectDefaultMetrics({ register: client.register, prefix: "arus_" });

metricsRouter.get("/metrics", async (_req, res) => {
  res.set("Content-Type", client.register.contentType);
  res.end(await client.register.metrics());
});
Mount it:

// server/index.ts
import { metricsRouter } from "./observability/metrics-route.js";
app.use("/metrics", metricsRouter);

4) Docker Compose (Grafana + Prometheus)

# docker-compose.yml
version: "3.8"
services:
  prometheus:
    image: prom/prometheus:v2.55.0
    volumes:
      - ./prometheus/prometheus.yml:/etc/prometheus/prometheus.yml
    ports: ["9090:9090"]
  grafana:
    image: grafana/grafana:11.2.0
    environment:
      - GF_SECURITY_ADMIN_USER=admin
      - GF_SECURITY_ADMIN_PASSWORD=admin
    volumes:
      - ./grafana/dashboards:/var/lib/grafana/dashboards
      - ./grafana/provisioning:/etc/grafana/provisioning
    ports: ["3000:3000"]

# grafana/provisioning/dashboards/dashboards.yml
apiVersion: 1
providers:
  - name: "arus"
    options: { path: "/var/lib/grafana/dashboards" }

# grafana/provisioning/datasources/prometheus.yml
apiVersion: 1
datasources:
  - name: Prometheus
    type: prometheus
    access: proxy
    url: http://prometheus:9090
    isDefault: true

Install Grafana locally, add Prometheus datasource pointing at http://localhost:9090, import grafana/dashboards/arus-rul.json.
