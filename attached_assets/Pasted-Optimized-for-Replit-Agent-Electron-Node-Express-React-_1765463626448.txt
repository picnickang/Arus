📘 SECTION 0 — GLOSSARY

Supplier — external vendor

Inventory Item — stored part/spare

UoM — unit of measure (pcs/L/m/kg/etc.)

ROB — Remaining On Board at PR creation

PR (Purchase Request) — internal request; generates grouped supplier emails

PO (Purchase Order) — created per supplier after PR is sent; tracks delivery

Line Item — requested part entry inside PR or PO
